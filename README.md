# NANCY Local Development Environment

Fully dockerized local setup for NANCY ERP. No AWS, no cloud dependencies.

## Quick start

See **[tasks/local-dev-setup-guide.md](tasks/local-dev-setup-guide.md)** for the complete step-by-step guide (Mac + Windows).

## Repo structure

```
local_nancy/
  docker/                     # Docker infrastructure
    docker-compose.yml        # Orchestrates all 10 containers
    Dockerfile.api            # .NET Core 3.1 API (built with SDK 6.0)
    Dockerfile.floorplan      # Python 3.10 + wkhtmltopdf
    Caddyfile                 # Reverse proxy + TLS (5 hostnames)
  services/
    auth-proxy/               # Replaces AWS Cognito (real DB auth, bcrypt, JWT)
    d365bc-mock/              # Replaces Microsoft Dynamics 365 Business Central
  config/
    appsettings.Local.json    # .NET API config (all cloud services → local Docker)
    .env                      # Shared secrets (DB, JWT, AES, MinIO)
    hosts-entries.txt         # /etc/hosts entries needed
  certs/                      # TLS certs (generate with mkcert, NOT committed)
    .gitkeep
  tasks/                      # Documentation
    local-dev-setup-guide.md  # Full setup guide (Mac + Windows)
    price-reconciliation-guide.md
```

## What's NOT in this repo

| Item | Why | Where to get it |
|---|---|---|
| TLS certificates | Machine-specific, generated by mkcert | `mkcert -install && mkcert dev.s10drd.com ...` |
| Database dump (~443 MB) | Too large for git, contains PII | Team lead (Jay / Robert) |
| Application repos (7) | Separate git repos at `laitkor-org` | See setup guide Step 1 |
